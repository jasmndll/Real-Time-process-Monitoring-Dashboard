# Real-Time OS Monitor

A graphical system dashboard built with Python, Tkinter, and Matplotlib.

## Features
- Live CPU, Memory, Disk, Network charts
- Per-core CPU usage bars
- Process table with kill, search, and filter
- 4 tabs: Overview, Processes, Disk, Network

## Run from source
pip install -r requirements.txt
python ui.py

## Requirements
Python 3.8+
